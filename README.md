# kyuren-2026-cross-domain-patent-torii

九州連合大会原稿用のLaTeXプロジェクトです。

本リポジトリは、卒業研究「特許文書の課題・解決手段分離に基づくクロスドメイン技術候補抽出」から、九州連合大会発表用に派生して作成したものです。

卒業研究計画書側では、提案手法の具体化と実現性検証としての位置づけを中心に整理します。一方、本リポジトリでは、実験結果のうち、特に U-Net の医療画像解析分野から製造欠陥検出分野への後年出現を参照した後ろ向き評価を中心に、短い研究発表原稿として整理します。

## Research Topic

本研究では、特許文書を `problem_context` と `solution_context` に分離し、課題が近い異分野間で、対象分野では低出現または未使用である技術手法・解決手段を、専門家が確認すべき候補として抽出する手法を検討します。

大会原稿では、主に次の点を扱います。

- 特許文書から課題文脈・解決手段文脈・技術手段を抽出する枠組み
- 医療画像解析分野で先行した U-Net を対象とした候補抽出
- 製造欠陥検出分野における後年出現を参照した後ろ向き評価
- 全文類似とは異なる根拠で候補を提示できる可能性

## Position of This Repository

このリポジトリは、卒業論文本体ではなく、九州連合大会原稿用の作業リポジトリです。

| リポジトリ | 役割 |
|---|---|
| 卒業研究本体リポジトリ | 研究計画書、卒論本文、実験全体の管理 |
| 本リポジトリ | 九州連合大会原稿、発表用図表、短縮版実験結果の管理 |

卒業研究計画書では、実験結果の詳細を前面に出しすぎず、提案手法の内容と検証の位置づけを中心に整理します。  
本リポジトリでは、U-Net後ろ向き評価や候補抽出結果を中心に、研究発表としての分かりやすさを重視して整理します。
